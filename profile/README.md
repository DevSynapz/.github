## 🧠 Our Mission & Focus

> *"Every idea is a node. DevSnapz connects them."*

We act as the synapse between developers, co-builders, and community support. DevSnapz is a local-first platform where developers can showcase their active projects, recruit collaborators, seek developer guidance, and collect sponsorships for hosting and API milestones.

### What You Can Do

| 🎯 Feature | 💡 Description |
| :--- | :--- |
| 🚀 **Project Showcase** | Register your web apps, AI pipelines, dev tools, and systems with custom gradients |
| 🤝 **Find Collaborators** | Post specific developer openings (e.g. Frontend React, Rust Systems) and collect applicant cover letters |
| 💡 **Guidance Board** | Share architectural questions and get targeted replies from mentors in the community |
| 🛡️ **Code Contributions** | Tag difficulty-labeled GitHub issues for open-source helpers to claim and merge |
| ☕ **Get Sponsored** | Define tiered support milestones and receive backing via credit card checkouts |

## 💻 Quick Start

Want to spin up the DevSnapz client locally? Follow these steps:

```bash
# 1. Clone the repository
git clone https://github.com/DevSnapz/Landing-1.git

# 2. Navigate to the project directory
cd Landing-1

# 3. Install dependencies
npm install

# 4. Spin up the development server
npm run dev
```

The database state (projects, guidance threads, backer lists) will persist automatically inside your browser's local storage.
